## what this ？

这是一个适用于 iOS 移动端的命令行工具集，提供常用自动自动化:

- [ ] x
- [ ] x
- [ ] x




## why use ruby ？

谁叫 cocoapods、fastlane … 都是 ruby 写的了 ~



## install

Add this line to your application's Gemfile:

```ruby
gem 'xzh'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xzh
